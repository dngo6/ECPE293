High Performance Computing 
Project 3: Gradient and Phase Images using Pthreads and OpenMP (Serial/Pthread/OpenMP)
David Duy Ngo

TO MAKE THE PROGRAMS:
Serial: make -f  makefile_serial
OpenMP: make -f makefile_omp
Pthread: make -f  makefile_pthread

TO RUN THE PROGRAMS:
Serial: ./serial <file_name> <sigma>
OpenMP: ./openMP <file_name> <sigma>
Pthread: ./pthread <file_name> <sigma> <num_threads>
